# Crypto Price Tracker (REST API + Python)

A simple Python script that fetches live cryptocurrency prices (INR + USD) using the CoinGecko REST API.

## Features
- Fetch real-time crypto prices
- Supports any coin (bitcoin, ethereum, dogecoin, etc.)
- REST API tested using Postman
- JSON parsing in Python
- Error handling + logging

## How to Run
python crypto_tracker.py

## API Used
https://api.coingecko.com/api/v3/simple/price?ids=bitcoin&vs_currencies=inr,usd
